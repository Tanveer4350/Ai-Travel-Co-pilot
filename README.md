# ✈️ AI Travel Copilot

Plan your perfect trip in seconds — powered by AI and tailored to your budget.

---

## 🚀 Overview

AI Travel Copilot is a full-stack AI-powered web application that generates personalized, budget-friendly travel itineraries based on user preferences.

Users can also modify their trips using natural language and view previously generated trips — making planning fast, interactive, and effortless.

---

## ✨ Features

- 🧠 AI-powered itinerary generation  
- 💰 Budget-aware trip planning  
- 📅 Day-wise structured plans  
- 💬 Modify trips using natural language  
- 🕘 Trip history (saved locally)  
- 🎨 Clean and interactive UI  

---

## 🛠️ Tech Stack

### Backend
- FastAPI (Python)
- Groq API (LLM)

### Frontend
- React.js
- CSS

### Other
- LocalStorage (history)
- REST API integration
